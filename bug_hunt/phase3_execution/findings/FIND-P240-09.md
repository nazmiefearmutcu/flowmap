### Repro
1. Construct merged `records` where first events are trades (common after time-warp interleaving).
2. Run alignment: bootstrap loop applies **all** snapshots first without resetting `local_book`.
3. Alignment pass starts with end-of-range book already in `local_book` / `last_known_mid`.
4. Early trades before the first in-order snapshot snap to **end-of-window** BBO/mid.

### Expected
Tracker starts empty; book state builds only from events ≤ trade time in the walk order. Bootstrap, if any, must not leave end-state in the same instance.

### Actual
```python
local_book = LocalBookTracker()
# bootstrap: apply EVERY snapshot in range → ends at last snapshot
for rec in records:
    if snapshot: apply_snapshot(...); last_known_mid = mid
# alignment uses same local_book (already last snapshot)
for rec in records:
    ...
```

### Fix hint
Remove bootstrap block, or `local_book = LocalBookTracker(); last_known_mid = None` after bootstrap if bootstrap is only for mid seed (seed mid without loading full end book into tracker used for rewrite).
