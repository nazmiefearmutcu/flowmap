### Repro (unit)

```python
from flowmap.engine.normalizer import AdaptiveNormalizer
import numpy as np

def live_refs(columns, fixed=3000.0):
    n = AdaptiveNormalizer(fixed_ref=fixed)
    refs, norms = [], []
    for col in columns:
        active = col[col > 0.01]
        if len(active):
            n.update(active)
        refs.append(n.global_ref)
        norms.append(n.normalize(col.copy()))
    return refs, norms, n.global_ref

def rebuild_ref(columns, fixed=3000.0):
    n = AdaptiveNormalizer(fixed_ref=fixed)
    # mimic rebuild: reset then ONE update over all active
    n.global_ref = fixed  # resets _initialized
    grid = np.stack(columns, axis=1)
    active = grid[grid > 0.01]
    if len(active):
        n.update(active)
    return n.global_ref, n.normalize(grid)

# Rising wall sizes over 50 columns
cols = [np.array([10.0 * (i+1), 5.0]) for i in range(50)]
refs_l, norms_l, ref_live = live_refs(cols)
ref_rb, norms_rb = rebuild_ref(cols)
# EXPECT contractual relation; ACTUAL large gap
assert abs(ref_live - ref_rb) < 1.0  # typically FAILS: live EMA trail ≠ one-shot p98
```

### Expected
Documented contract: same history → same (or ε-close) norm/ref, or intentional "recolor on rebuild" UX.

### Actual
| Path | Ref update | History colors |
|------|------------|----------------|
| Live | Per-column EMA p98 α=0.05 | Past columns **frozen** at old LUT indices |
| Rebuild | Reset to `config.bid_ref` then **one** `update(all active cells)` | Entire buffer recolored with batch ref |

Also smooth diverge: live 1D `np.convolve` (`_smooth_column`); rebuild SciPy `gaussian_filter1d` axis=0.

User symptom: resize / scroll / ticks_per_row zoom triggers rebuild → global color jump of history.

### Fix hint
Shared pipeline: either (A) rebuild without ref reset + sequential update schedule, (B) two-pass freeze ref from full history then paint, (C) document recolor-as-feature. Unify smooth kernel.
