### Repro
1. Construct column where `bid_arr[r] == ask_arr[r] > 0` and `mid_price == 0` (no BBO).
2. Live path: `is_bid = bid_arr > ask_arr` → **False** for equal sizes → ask LUT.
3. Rebuild path: `is_bid = bid_grid >= ask_grid` → **True** for equal sizes → bid LUT.

### Expected
Identical side decision on live and rebuild for equal bid/ask sizes.

### Actual
| Path | Operator | Equal sizes side |
|------|----------|------------------|
| Live `_draw_column` | `bid_arr > ask_arr` | ask |
| Rebuild | `bid_grid >= ask_grid` | bid |

Different BOOKMAP LUT family on resize/rebuild for no-mid / zero-mid frames.

### Fix hint
Unify to one comparator (`>=` or `>` both sides) shared helper `side_mask(bid, ask, mid)`.
