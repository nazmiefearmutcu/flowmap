### Repro
1. Focus outside heatmap (or event handled by MainWindow): `+`/`-` → `heatmap.zoom_in/out` → **row_height**.
2. Focus heatmap (StrongFocus): same keys → `price_zoom_in/out` → **ticks_per_row** (Ctrl+ → row_height, Shift+ → timeframe).
3. User cannot see which focus target owns the binding; accidental ticks collapse changes price scale radically.

### Expected
Single documented map (global QShortcut) or visible focus chrome + consistent binding.

### Actual
Silent dual semantics; high risk of “heatmap jumped” reports after clicking chart then pressing +.

### Fix hint
Unify on one zoom ladder for bare +/-; keep modifiers for secondary axes; document matrix in status help.
