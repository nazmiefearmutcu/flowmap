### Repro
1. Enable CCXT WS path (`_WsWorker`).
2. `_sender_loop` emits BBO from order book snapshot **and** from ticker when each nonce changes.
3. Mid/BBO can thrash between slightly different sources at ~33 Hz.

### Expected
Single authoritative BBO (prefer L2 top-of-book); ticker optional fallback only when book absent.

### Actual
Both paths emit `("bbo", bbo)` / `sig_bbo` independently after nonce fix.

### Fix hint
Emit ticker BBO only if no OB within N ms; or drop ticker BBO entirely when watching order book.
