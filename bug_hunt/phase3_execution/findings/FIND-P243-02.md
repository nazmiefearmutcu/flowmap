### Repro
1. Scroll history so `_scroll_offset > 0`.
2. Press **R** → `reset_view()` sets `row_height=4`, `column_width=1.0`, `auto_follow=True`, calls `rebuild_heatmap()`.
3. Rebuild slices history with `end_idx = len(history) - self._scroll_offset` — still scrolled.
4. Status: "View Reset: auto-follow ON, default zoom" implies return to live defaults.

### Expected
Reset clears scroll to live edge (`_scroll_offset=0`) like Go Live + default zoom.

### Actual
Offset retained through rebuild; follow flag true but history window until next push zeros offset.

### Fix hint
`self._scroll_offset = 0` before rebuild in `reset_view`.
