### Repro
1. Start Crypcodile live feed.
2. Force WS disconnect / connector exception (kill network, invalid stream).
3. Observe `sig_error` / `sig_disconnected`; feed never recovers without full provider reconnect from UI.
4. `_run` does single `await connector.run()` then exits worker start path.

### Expected
Backoff reconnect loop while provider still “connected”; preserve symbol/channels; emit status.

### Actual
```python
try:
    await connector.run()
except asyncio.CancelledError:
    raise
except Exception as e:
    self.sig_error.emit(f"Connector run error: {e}")
finally:
    self.sig_disconnected.emit()
# no retry / re-make_connector
```

### Fix hint
```python
while self._running:
    try:
        await connector.run()
    except CancelledError: raise
    except Exception as e:
        emit error; await asyncio.sleep(backoff)
        recreate connector/transport
```
