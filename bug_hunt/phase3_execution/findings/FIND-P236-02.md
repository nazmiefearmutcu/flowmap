2. Synthetic case: mid-bin place large wall, remove wall before bin end; only end-of-bin book is painted.
3. Compare to standalone `CrypcodileReplayProvider` which emits event-level columns via `_dispatch_record`.

### Expected
Documented lossy “preview” mode **or** event-faithful / max-depth-per-bin product mode; users not misled that CLI hist equals tape replay.

### Fix hint
