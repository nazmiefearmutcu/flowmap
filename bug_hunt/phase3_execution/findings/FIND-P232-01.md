### Repro
1. Core provides `is_buy_side()` treating BUY and BID as buy (`core/__init__.py:19-28`).
2. Call `VolumeBubbles.add_trade(p, s, Side.BID, tick)` → size goes to `sell_size` (else branch).
3. Call with `Side.BUY` → `buy_size`; `Side.SELL` → `sell_size`; non-SELL non-BUY also zero both on new bubble path (`sell_size` only if exact SELL).
4. Pulse: `delta = size if side == Side.BUY else -size` → BID flips CVD negative.

### Expected
Shared `is_buy_side` / `is_sell_side` for all trade overlays.

### Actual
BUY-only equality; BID aggressor (if any path emits it) misclassified; order_book CVD can disagree with pulse/bubbles colors.
