### Repro
1. During `load_plugin`, parent dir is `sys.path.insert(0, …)` while `exec_module` runs — plugin can ship `json.py` / `flowmap.py` and shadow imports for concurrent loaders.
2. `AddonState.get_order_book()` returns live `OrderBook` reference, not a read-only view — plugin can mutate book maps and corrupt the heatmap.
3. `register_with_app` reassigns `ob.on_trade` / `ob.on_bbo` wrappers; second plugin registration can nest or clobber; no unwind API.

### Expected
No sys.path mutation (absolute imports only). Read-only `OrderBookView` Protocol. Callback list append, not reassignment.

### Actual
Shadowing + mutation + clobber vectors are latent until wire; they are design defects in the API as shipped.

### Fix hint
Force `spec` load without path insert; freeze or copy book state; multi-plugin callback chain with remove.

### Evidence
- `loader.py:91` insert/pop path
- `plugin_api.py:238` `return self._api.get_order_book()`
